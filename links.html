Links

Credits and useful links/resources I've found from other websites or on my own. Graphic credits/resources in graphics. This pages is a constant work in progress.

Credits

  • Lightbox2: Free to use plugin. I use this for my galleries like in the creatives pages.
  • Juneish: Where I learnt to make pure CSS tabs. Not the most elegant solution in my opinion but definitely the simplest.
  • petrapixel: status.cafe widget I used on my homepage.
  • foollovers: The navigation background is a modified background from here. It has materials like stickers, banners, backgrounds, etc.
  • da-lace: Lace border. Lots of lace here. It's in the name.

Other Links

For Webmasters

  • w3schools: A place to start learning basics of various coding languages such as HTML, CSS, and JavaScript. More of a stepping stone and should not be the only learning source.
  • xandra on 32bit.cafe: Resources masterlist for the personal web.
  • petrapixel: A lot of things. Mainly related to web development, especially for beginners and those using Neocities. I followed her GitHub tutorial.
  • stackoverflow: Forum for developers, good for specific questions. Usually, I just Google my question and try to find a similar thread on there.
  • wave.webaim: Accessibility checker.
  • webaim: Contrast checker for accessibility.
  • screenfly: Test your site on multiple screen sizes.
  • scripted: Code snippets.
  • brisray: Webring masterlist.

Visuals

  • thetextures: Textures.
  • dafont: Font website. I get mine from here.
  • da-lace: Free to use lace materials.

Unsorted

  • papercraft-maker: Papercraft viewer, like an online Pepakura. It can open .pdo files, among other file types. I wish I discovered this ealier into my papercrafting phase.
  • photopea: Online Photoshop
  • iloveimg: My preferred image resizer.
  • imgonline: Cut an image into equal pieces.
  • lolitahistory: Lolita fashion magazine scans.
